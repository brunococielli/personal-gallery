A personal image gallery application that allows users to create an account, authenticate, and manage their own private image collection.  
The project focuses on secure authentication, backend logic, and user-specific data handling.

## Features

- User registration and login
- Authentication and protected routes
- Private galleries per user
- Image upload and management
- Email sending via external API
- Input validation and basic security measures

## Tech Stack

- Node.js
- Express
- Prisma ORM
- Neon (PostgreSQL)
- Authentication 
- Email API integration
- JavaScript / HTML / CSS

## How It Works

- Users can create an account and log in securely
- Authentication protects private gallery routes
- Each user can only access their own images
- Image data is stored and linked to the authenticated user
- Emails are sent using an external API for account-related actions
- Prisma handles database queries and relationships
